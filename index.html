🛡️ راهنمای mhrv-rs + Tunnel Node

دور زدن رایگان DPI با Google Apps Script + تانل کامل VPS — برای اندروید

✓ رایگان Android Full Tunnel VPS بدون ریشه

📱 این آموزش برای اندروید است. برای راه‌اندازی اولیه به یک فیلترشکن فعال نیاز داری — بعد از تنظیم کامل دیگر به هیچ چیز دیگری نیاز نخواهی داشت.

🔑 چرا کار می‌کند؟ این روش ترافیک تو رو از طریق سرورهای Google (که فیلتر نیست) رد می‌کند. ISP فقط اتصال به www.google.com می‌بیند و قادر به تشخیص یا فیلتر کردن نیست — تا زمانی که گوگل در ایران فیلتر نشود.

🖥️ چرا VPS؟ بدون VPS فقط HTTP relay داری و اپ‌هایی مثل تلگرام که TCP خام استفاده می‌کنند کار نمی‌کنند. با VPS یه Tunnel Node اجرا می‌شه که همه ترافیک TCP — تلگرام، SSH، بانک و... — از طریقش رد می‌شه.

👥 اشتراک VPS: یک VPS می‌تواند برای چند نفر استفاده شود — کافیه هر نفر CodeFull.gs رو با جیمیل خودش deploy کنه و ID بگیره. نیازی نیست به VPS دسترسی داشته باشند.

👥 چند Deployment — سرعت بیشتر

    
💡 هر اکانت گوگل = یک Deployment = ۳۰ اتصال همزمان. برای سرعت بیشتر باید اکانت‌های مختلف استفاده کنی.


    
چرا چند مرورگر؟ هر مرورگر می‌تواند با یک جیمیل جداگانه login باشد. این‌طوری می‌توانی روی یک گوشی چند اکانت مختلف داشته باشی و از هر کدام یک Deployment ID بگیری.


    

      
1

      
چند مرورگر مختلف نصب کن
Chrome، Brave، Firefox یا Edge — هر کدوم برای یه جیمیل جداگانه

    

    

      
2

      
در هر مرورگر با یه جیمیل جدید وارد شو
به script.google.com برو → CodeFull.gs رو deploy کن → ID بگیر

    

    

      
3

      
همه ID ها رو به اپ اضافه کن
در بخش Deployment IDs — هر ID در یک خط جداگانه. اپ round-robin استفاده می‌کنه.

    


    
🔐 مهم: از جیمیل اصلی و شخصیت استفاده نکن. ریسک محدودیت یا تعلیق اکانت وجود دارد. جیمیل‌های جدید بساز.


    
👥 اشتراک با دیگران: یک VPS می‌تواند بین چند نفر مشترک باشد. کافیه هر نفر CodeFull.gs رو با جیمیل خودش deploy کنه و ID بگیره — نیازی به دسترسی به VPS ندارند. فقط کد CodeFull.gs رو بهشون بده که deploy کنن.
